= Panoramica di NetApp Disaster Recovery
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/use/


[role="lead"]
Utilizzando NetApp Disaster Recovery, è possibile raggiungere i seguenti obiettivi:

* link:../use/dashboard-view.html["Visualizza lo stato di salute dei tuoi piani di disaster recovery"]
* link:../use/sites-add.html["Aggiungere risorse a un sito"]
* link:../use/resource-group.html["Crea gruppi di risorse per organizzare le fonti di dati"]
* link:../use/drplan-create.html["Creare un piano di ripristino in caso di disastro"]
* link:../use/replicate.html["Replicare le app VMware"] dal tuo sito primario a un sito di disaster recovery remoto nel cloud utilizzando la replica SnapMirror
* link:../use/migrate.html["Migra le applicazioni"] dal tuo sito principale a un altro sito
* link:../use/failover.html["Testare il failover"] senza interrompere le risorse
* In caso di disastro, link:../use/failover.html["esegui il failover del tuo sito primario"]
* Dopo che il disastro è stato risolto, link:../use/failback.html["fallire indietro"] dal sito di disaster recovery al sito principale
* link:../use/monitor-jobs.html["Monitorare le operazioni di ripristino in caso di disastro"] nella pagina Monitoraggio dei lavori
