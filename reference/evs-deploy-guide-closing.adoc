= Conclusione
:allow-uri-read: 


[role="lead"]
NetApp Disaster Recovery fornisce una soluzione DR ibrida e conveniente per l'infrastruttura VMware on-premise, se configurata per utilizzare Amazon EVS e Amazon FSx for NetApp ONTAP come destinazione DR.  Ciò fornisce un obiettivo DR affidabile per infrastrutture mission critical.  NetApp Disaster Recovery sfrutta le funzionalità native di protezione dei dati ONTAP , supportate da decenni di esperienza nella protezione dei dati e ampiamente testate per garantire la completa protezione dei dati.

Per maggiori informazioni:

* https://docs.aws.amazon.com/fsx/latest/ONTAPGuide/getting-started.html["Introduzione ad Amazon FSx for NetApp ONTAP"^]
* https://docs.netapp.com/us-en/bluexp-disaster-recovery/index.html["Documentazione NetApp Disaster Recovery"]
